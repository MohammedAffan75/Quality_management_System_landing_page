# CMTI's Quality Management System (QMS)

![QMS Banner](./image.png)

A premium, enterprise-grade SaaS landing page for **CMTI's Quality Management System**. This platform is designed to modernize engineering quality workflows by digitalizing drawings, automating deviation monitoring, and integrating wireless measurement instruments into a secure, cloud-centric ecosystem.

## 🚀 Key Features

- **📐 Dimension Ballooning**: Automate Bill of Characteristics extraction from 2D engineering drawings with sub-pixel precision.
- **📊 Real-time Deviation Monitoring**: Instant comparison of measurement data against engineering specifications.
- **📡 Wireless Hardware Integration**: Seamless connectivity with Bluetooth and BLE inspection instruments for error-free data collection.
- **📝 Automated Reporting**: Generate professional-grade inspection reports (AS9102, ISO 9001, IATF 16949) with a single click.
- **🛡️ Compliance Ready**: Built-in workflows tailored for aerospace and automotive standards.
- **☁️ Cloud-Native Infrastructure**: Secure, real-time data synchronization accessible from any authorized device.

## 📈 Performance Impact

Our platform delivers measurable results for manufacturing leaders:

- **98%** Efficiency Gain in data collection.
- **85%** Reduction in human transcription errors.
- **30%** Time Saved on overall inspection reporting.
- **10k+** Reports generated daily across our network.

## 🛠️ Technology Stack

- **Frontend**: Standard-compliant HTML5, Modern Vanilla CSS3 (Custom properties, Grid, Flexbox), and Responsive JavaScript.
- **Aesthetics**: Glassmorphism UI, smooth scroll-triggered animations (Intersection Observer API), and premium typography using the Inter font family.
- **Icons**: Emoji-based iconography for a clean, modern look.

## 📦 Getting Started

To view the landing page locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/MohammedAffan75/Quality_management_System_landing_page.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Quality_management_System_landing_page
   ```
3. Open `index.html` in your preferred web browser.
